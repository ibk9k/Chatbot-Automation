Widget Sandbox Page

This page simulates a client's website embedding your multi-client chat widget. In the bottom-right corner, you should see the chat bubble button.

Quick Setup & Verification Checklist:

  1. Make sure your Supabase project is active and you have run the schema in `supabase/schema.sql`.
  2. Ensure your environment variables are configured (`OPENAI_API_KEY`, `SUPABASE_URL`, `SUPABASE_SERVICE_KEY`) in a `.env.local` file in your project folder.
  3. Start the local server by running npm run dev.
  4. Open this page at http://localhost:3000/test-widget.html.
  5. Click the chat bubble in the bottom right, send a message, and check the stream!
  6. Change the monthly_limit to 2 in your Supabase clients table for test-client, then verify the third message gets blocked with a "limit reached" response.

Snippet embedded in this page:

<script src="/widget.js" data-client-id="test-client"></script>
